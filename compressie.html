COMPRESSIE

Compressie betekent dat je code kleiner maakt, zodat het minder plek inneemt en sneller werkt. Dit doe je door bijvoorbeeld lege ruimtes, nieuwe regels en lange namen korter te maken. De code ziet er daarna moeilijker uit, maar werkt nogsteeds hetzelfde. Dit wordt vaak gedaan bij websites of apps, zodat ze sneller laden en minder internet gebruiken. Het helpt de computer, maar het maakt de code wel moeilijker om te begrijpen.
